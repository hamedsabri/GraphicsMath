inline

Compute the intersection(s) between a ray and an implicit sphere.

The points of intersection can be computed from the intersection magnitudes o_intersections /// via RayPosition.

Example usage:

if ( gm::RaySphereIntersection( sphereOrigin,
sphereRadius,
rayOrigin,
rayDirection,
intersections ) == 2 )
{
gm::Vec3f firstIntersectionPoint = gm::RayPosition( rayOrigin, rayDirection, intersections.Min() );
// ...
}
Parameters
i_sphereOriginThe origin or center of the sphere.
i_sphereRadiusThe radius of the sphere.
i_rayOriginThe origin of the ray.
i_rayDirectionThe direction of the ray.
o_intersectionsThe ray magnitudes of the intersections.
Returns
The number of times the ray intersections the sphere.
Return values
0The ray does not intersect the sphere at all.
1The ray intersects the surface of the sphere exactly.
2the ray intersects through the volume of the sphere.
