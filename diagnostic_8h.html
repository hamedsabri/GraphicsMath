Asserts that the expression expr is true in debug builds.

If expr evalutes false, an error message will be printed with contextual information including the failure site.

In release builds, this is compiled out.

Verifies that the expression expr evaluates to true at runtime.

If expr evalutes false, an error message will be printed with contextual information including the failure site.

GM_VERIFY is different from GM_ASSERT in that it does not get compiled out for release builds, so use sparingly!
